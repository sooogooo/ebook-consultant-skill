# 医美咨询师实战技巧网站

## 项目简介

这是一个现代化的医美咨询师培训网站，提供AI智能助手、课程内容、设计工具等功能。网站采用响应式设计，支持手机、平板、PC多端访问。

## 功能特色

### ✨ 核心功能

1. **AI智能助手**
   - 智能问答系统
   - 自动生成问题建议
   - 对话历史记录
   - 选词AI功能（划词解释、展开说明）
   - 多种输出风格：轻松幽默、标准日常、科学严谨
   - 多种输出长度：详细、标准、简约

2. **课程内容**
   - 16章完整课程体系
   - 分类展示，可折叠展开
   - 一键访问课程详情
   - 移动端优化

3. **设计工具**
   - Canvas绘图功能
   - 多种绘图工具：画笔、橡皮擦、直线、圆形、矩形
   - 颜色和粗细调节
   - 文字添加功能
   - 导出PNG/PDF格式
   - iOS设备完美兼容

4. **主题设置**
   - 5种优雅配色：优雅米白、薄荷清新、淡紫梦幻、玫瑰金、海洋蓝
   - 3种字体大小：小、中、大
   - 设置本地存储
   - 实时预览效果

### 📱 移动端优化

- 手机优先的响应式设计
- iOS设备完美兼容（ специаль处理触摸事件）
- 触摸友好的交互设计
- 流畅的动画效果

### 🎨 视觉设计

- 中性优雅的设计调性
- 现代Material Design风格
- 大量使用SVG图标
- 细线条简约设计
- 柔和的色彩搭配

## 技术栈

- **HTML5** - 语义化标签
- **CSS3** - 自定义属性、Grid、Flexbox
- **JavaScript ES6+** - 模块化、类、Promise

## 使用说明

### 本地开发

1. 直接用浏览器打开 `index.html`
2. 或使用本地服务器：
   ```bash
   python -m http.server 8080
   ```
   然后访问 `http://localhost:8080`

### 功能使用

#### AI智能助手
1. 在输入框输入问题
2. 点击发送按钮或按回车键
3. 查看AI回复
4. 点击建议问题快速提问
5. 查看对话历史记录
6. 选中文本使用选词AI功能

#### 设计工具
1. 选择绘图工具
2. 调整颜色和粗细
3. 在画布上绘制
4. 添加文字或图形
5. 导出为PNG或PDF

#### 设置
1. 点击右上角设置按钮
2. 选择喜欢的主题配色
3. 调整字体大小
4. 设置AI输出风格和长度
